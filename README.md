# TOK Corporate Wellness Retreat 2026 — Flyer

Professional event flyer for Temple of Knowledge's corporate wellness retreat.

## Setup

1. Create folder structure:
```
project/
├── Flyer1_Working.html
└── img/
    ├── header-hero.jpeg
    ├── footer-hero.jpeg
    └── logo.png
```

2. Open `Flyer1_Working.html` in browser

## Files Required

- `header-hero.jpeg` — Header background (176 KB)
- `footer-hero.jpeg` — Footer background (168 KB)  
- `logo.png` — Logo image (7.8 KB)

## Customization

Edit text in HTML:
- Event date, time, venue (line ~525)
- Agenda items (lines ~565-577)
- Highlights (lines ~584-596)
- Contact numbers (lines ~677-679)

Edit colors in CSS (`:root` variables):
- `--cream` — Background
- `--saffron` — Accent color
- `--gold-pale` — Highlight color

## Browser Support

Chrome, Firefox, Safari, Edge (latest versions)
